# TCC
Classificação de Línguas e Etnias Indígenas nos Censos Demográfico e Agropecuário por meio de Machine Learning

Os códigos estão divididos em classificação de Etnias e de Línguas Indígenas.

Executá-los conforme a numeração, de acordo com a classificação desejada.

Os arquivos de leitura deverão ser copiados para o mesmo diretório dos códigos:

- BDD_etnias_20250505.xlsx
- BDD_linguas_20240604.xlsx
- descritores.xlsx
- indigenas.csv (amostra de microdados, sintéticos em consonância com a LGPD e a Lei do Sigilo Estatístico)
